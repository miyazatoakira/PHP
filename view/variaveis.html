Variáveis em PHP

Ir para os exercícios

1. Definição:
Em PHP, uma variável é uma representação simbólica de um valor. Variáveis são utilizadas para armazenar dados que podem ser modificados durante a execução do programa.

2. Declaração de Variáveis:
As variáveis em PHP começam com o símbolo $, seguido pelo nome da variável. Os nomes das variáveis em PHP são sensíveis a maiúsculas e minúsculas.
Exemplos: 

<?php
  $nome = "Ana";
  $idade = 28;
    echo "Olá, meu nome é $nome e eu tenho $idade anos.";
?>

3. Tipos de Variáveis:
Em PHP, as variáveis podem armazenar diferentes tipos de dados, como strings, números, booleanos, arrays, objetos, etc. PHP é uma linguagem de tipagem dinâmica, o que significa que não é necessário declarar explicitamente o tipo de uma variável antes de usá-la.
Exemplos: 

<?php
$nome = "Maria"; // string
$idade = 30; // integer
$altura = 1.75; // float
$casado = true; // boolean
$cores = array("vermelho", "verde", "azul"); // array
?>

4. Escopo de Variáveis:
O escopo de uma variável refere-se à sua acessibilidade dentro do programa. Existem três escopos de variáveis em PHP: global, local e estático. As variáveis globais podem ser acessadas de qualquer parte do script PHP. As variáveis locais são definidas dentro de uma função e só podem ser acessadas dentro dessa função. As variáveis estáticas mantêm seu valor entre chamadas de função.

O que são
Estruturas de Repetição ?

Neste trabalho exploraremos o que são Estruturas de Repetição presentes na Linguagem de Programação PHP, além de trazer inúmeros exemplos de uso.

Ao desenvolver aplicações, desde sites simples até sistemas complexos, as estruturas de repetição são recursos fundamentais. Elas nos permitem lidar com uma variedade de tarefas, desde exibir grandes volumes de dados para os usuários finais até resolver problemas matemáticos, como a sequência de Fibonacci.

No PHP, existem várias maneiras de criar loops de repetição. Podemos utilizá-los para iterar um determinado número de vezes ou para percorrer estruturas de dados como arrays e objetos.

Essas estruturas nos proporcionam flexibilidade e eficiência na manipulação e processamento de dados, tornando o desenvolvimento de aplicações mais dinâmico e poderoso.

Estrutura FOR

Uma das Estruturas de Repetição mais utilizadas e presente em quase todas as Linguagens de Programação. Sendo usado mormente para iterar definir loops e iterar arrays.

A estrutura de repetição "for" é a mais comumente utilizada durante o desenvolvimento em PHP. Ela permite a criação de loops que vão de um ponto de partida a outro pré-definido, tudo dentro da sintaxe do próprio loop. Além disso, sua sintaxe é altamente flexível, destacando-se como a mais versátil dentre todas as estruturas de repetição do PHP. Basicamente, sua estrutura é composta por três expressões, como exemplificado abaixo:

Loop Básico:

Exemplo básico do uso do FOR, com a primeira entrada de valor representando o INÍCIO do Loop, a segunda o FIM do Loop e o terceiro o INCREMENTO do LOOP

php

for ($i = 0; $i < 5; $i++) {
echo "Sintaxe Básica"
}

Loop Infinito:

Sendo bem sugestivo, recomenda-se precaução antes de usar esse tipo de Estrutura !

php

for (; ;) {
echo "Loop Infinito !"
}

Loop for em Iteráveis: Iteração em Arrays

Um dos usos mais frequentes do FOR, sendo o uso do mesmo para percorrer ARRAYS

php

$nomes = ["Alice", "Bob", "Charlie"];
foreach ($nomes as $nome) {
echo $nome;
}

Neste exemplo, $nomes é o array que você deseja percorrer, e $nome é uma variável que irá conter cada elemento do array em cada iteração.

Estrutura While

Outra estrutura de repetição presente no PHP e muitas outras Linguagens é o While, o qual permite o início de um LOOP até que a CONDIÇÃO = TRUE.

O while é uma estrutura de controle em PHP (e em muitas outras linguagens de programação) usada para repetir um bloco de código enquanto uma condição específica é verdadeira. Ele executa o bloco de código repetidamente até que a condição especificada seja avaliada como falsa.

php

while(condicao){
echo "Olá Mundo";
}

Do-While

Uma forma alternativa do uso do WHILE se dá com o uso do DO.

php

$i = 0;
do{
// Bloco de código a ser repetido
$i++;
}
while ($i < 10);

É semelhante ao while, mas a verificação da condição ocorre após a execução do bloco de código, garantindo que o bloco de código seja executado pelo menos uma vez, mesmo se a condição inicialmente for falsa. Como a demonstrada acima.

Break e Continue

Formas adicionais para controle de seus loops

php

}

     echo $i;
     continue;
} }

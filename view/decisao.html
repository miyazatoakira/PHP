O que são estruturas de decisão ?

A estrutura da tomada de decisão na programação refere-se à capacidade de um programa de fazer escolhas lógicas com base em condições específicas. É uma parte fundamental da lógica de controle em muitas linguagens de programação, permitindo que um programa execute diferentes blocos de código dependendo do resultado de expressões booleanas (verdadeiro ou falso).

Existem várias formas comuns de estruturas de tomada de decisão na programação, tais como:

Declarações if:



O laço if é uma estrutura de controle fundamental em programação que permite que o programa tome decisões com base em condições específicas. Basicamente, o if permite, por exemplo que você avalie uma expressão booleana (verdadeira ou falsa) e execute um bloco de código se essa condição for verdadeira. O laço if permite também que o usuário possa verificar outros tipos de condições como equidade entre valores, qual valor é maior, casos de duas ou mais condições verdadeiras em uma mesma estrutura de decisão, e vários outros casos.

if: Uma declaração if é usada para executar um bloco de código se uma condição especificada for verdadeira. Ela geralmente é escrita na forma:


php

$idade = 20; if ($idade >= 18) { echo "Você é maior de idade"; }


if-else: A declaração if-else é usada para executar um bloco de código se a condição especificada for verdadeira e outro bloco de código se a condição for falsa. É escrita na forma:


php

$idade = 20; if ($idade >= 18) { echo "Você é maior de idade"; } else { echo "Você é menor de idade"; }


if-elif-else: Esta estrutura é uma extensão da if-else e permite testar múltiplas condições. É escrita na forma:


php

$nota = 75; if ($nota >= 90) { echo "A"; } elseif ($nota >= 80) { echo "B"; } elseif ($nota >= 70) { echo "C"; } else{ echo "D"; }




Laço Switch-case (ou equivalente):






O laço switch-case é uma estrutura de controle que permite avaliar uma expressão e, dependendo do valor dessa expressão, executar diferentes blocos de código. É uma maneira eficiente de lidar com múltiplas condições em comparação com múltiplos if-else if-else. A estrutura básica de um laço switch-case em muitas linguagens de programação, incluindo Java, C, C++, PHP e outras, é a seguinte:



Exemplo Básico:

php

$dia = 3; $diaDaSemana = ""; switch ($dia) { case 1: $diaDaSemana = "Domingo"; break; case 2: $diaDaSemana = "Segunda-feira"; break; case 3: $diaDaSemana = "Terça-feira"; break; case 4: $diaDaSemana = "Quarta-feira"; break; case 5: $diaDaSemana = "Quinta-feira"; break; case 6: $diaDaSemana = "Sexta-feira"; break; case 7: $diaDaSemana = "Sabado"; break; default: $diaDaSemana = "Dia inválido"; } echo "Hoje é " . $diaDaSemana;


Explicação

A expressão dentro dos parênteses após a palavra-chave switch é avaliada. O valor dessa expressão é então comparado com cada caso (case) na estrutura. Se o valor da expressão for igual a um dos valores dos casos, o bloco de código correspondente a esse caso é executado. A palavra-chave break é usada para sair do laço switch-case após a execução do bloco de código correspondente a um caso. Isso evita que os blocos de código dos casos subsequentes sejam executados erroneamente. Se o valor da expressão não corresponder a nenhum dos casos listados, o bloco de código dentro do default é executado, se estiver presente. O default é opcional.



Exemplo com múltiplas condições em um caso:

php

$nota = 75; switch (true) { case ($nota >= 90); echo "MB"; break; case ($nota >= 80); echo "B"; break; case ($nota >= 60); echo "R"; break; default: echo "I"; }

Neste último exemplo, a expressão true é usada no switch, e cada caso contém uma condição booleana que é avaliada. Isso permite que você use múltiplas condições em um único caso.
